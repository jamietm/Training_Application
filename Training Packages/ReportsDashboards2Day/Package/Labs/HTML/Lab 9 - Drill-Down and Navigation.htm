Lab Overview

 

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

         We are now going to look at navigating from one Analysis to another

         The aim will be to navigate from our Order Revenue by Quarter Analysis to view the Sales Order Details Analysis

         In order for the navigation to work, we will first create a Saved Filter containing all the possible Is Prompted filters that we may need. We will then apply this Saved Filter to our target Sales Order Details Analysis

 

 

 

Information

 

         We will implement the navigation using a named Action. This means that we will have an Action stored in the Presentation Catalog which can be re-used across many Analyses

 

         Once we have tested the navigation works, we will then make the Action conditional so that it can only be initiated if there are less than 250 order records to navigate to (this will make sure nobody impacts performance by downloading large sets of results)

 

 

 

 

 

 

Create Is Prompted Saved Filter

 

 

Peak Indicators Limited

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

Create Saved Filter

 

      Create a new Filter using the Sales Orders Subject Area

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Create Is Prompted Filters

 

      Create Is Prompted filters for the following columns:

  Time.Year

  Time.Quarter

  Time.Month

  Customer.Customer Class Code

  Customer.Sales Channel Code

  Organization.Business Group

  Organization.Organization Name

  Order.Deal Size

 

 

 

 

 

 

 

 

 

 

 

 

 

Save Filter

 

         Save the filter in your My Folders area:

 

  Folder: /My Folders/Subject Area Contents/Sales Orders

 

  Name: Is Prompted Filters

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Apply Saved Filter

 

 

Peak Indicators Limited

 

 

 

 

 

Information

 

         Now that we have created our Is Prompted Saved Filter, we need to apply it to our target Analysis Sales Order Details

         Open the following Analysis for editing:

 

  Folder: My Folders

 

  Name: Sales Order Details

 

 

 

 

 

 

 

 

 

 

 

Removing Existing Filters

 

         On the Criteria tab, remove all the existing filters that are on the Analysis

  You can do this by clicking on the button in the Filters pane

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Apply Saved Filter

 

         Within the Catalog pane on the Criteria tab, browse for your Saved Filter, you will find it in the following location:

 

  Folder: /My Folders/Subject Area Contents/Is Prompted Filters

  Name: Is Prompted Filters

 

 

 

 

 

 

 

 

 

 

 

 

 

Save Analysis

 

         We have now applied our Is Prompted filters to the target Analysis

 

         Save the Analysis

Configure Navigation

 

 

Peak Indicators Limited

 

 

 

Information

 

         We will now edit our Order Revenue by Quarter Analysis and add an Action Link to the Total Revenue (Millions) column

         The Action Link will be configured to initiate the View Sales Order Details Action which we just created

         Because we have a Column Selector on this Analysis, well configure the Column Properties for Total Revenue (Millions) within the Column Selector view, normally we would be doing this on the Criteria tab

 

 

 

 

Open Analysis

 

         Open the following Analysis for editing:

 

  Folder: My Folders

 

  Name: Order Revenue by Quarter

 

 

 

 

 

 

 

 

 

 

 

Column Properties

 

         Edit the Column Selector view on the Compound Layout

 

         Open up the Column Properties for the Total Revenue (Millions) column

 

         Go to the Interaction tab

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Create Action Link

 

         Set the Value Primary Interaction setting to: Action Links

 

         Add a new Action Link

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Configure Action Link

 

         Specify the following Link Text: View Sales Order Details

 

         Click the button and select the following Action:

 

  Folder: My Folders

 

  Name: View Sales Order Details

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Test Action Link

 

         Your Action Link is now configured!

         Return back to the Compound Layout and save your Analysis

         Preview your Analysis and test the navigation

  Test the navigation by clicking on a Total Revenue (Millions) figure and choosing the View Sales Order Details option

  You should see a confirmation window appear

  Check the Filters view at the bottom to make sure the correct filters are being applied to the target Analysis!

 

 

 

 

 

 

 

 

 

Apply Condition

 

         Edit the Column Selector view once again and open up the Column Properties for Total Revenue (Millions)

 

         Go to the Interaction tab and open up your Action Link for editing

 

 

 

 

 

 

 

 

 

 

 

 

Apply Condition

 

         Apply the following Condition for the Action Link:

 

  # Orders is less than 250

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Test Condition

 

         Return back to the Compound Layout and save the Analysis

 

         Preview the Analysis and test the navigation once again

 

         Notice this time that many of the measure values are no longer hyperlinked as they do not meet the Action Link criteria!

Peak Indicators Limited

 

 

 

 

 

 

 

 

Information

 

         As a final and very simple test, we are going to combine both Drill-Down and Navigation

         The aim is to test that Navigation works properly even after we drill-down to Month and Organization Name levels on the Analysis

         When the target Analysis is opened, we shall check the Filters view to confirm that the correct filter criteria have been applied

 

 

 

 

Drill-Down and Navigation

 

         Preview your Analysis once again

 

         Drill-down on a Business Group e.g. Vision West Europe

 

         Drill-down on a Quarter e.g. 2007 Q 1

 

 

 

 

 

 

 

 

 

Test Navigation

 

         Now attempt to navigate on one of the Measure values e.g. where Month = 2007 / 01 and Organization Name = Vision Switzerland

 

         You should find that the target Sales Order Details Analysis applies the correct filters selections. For example:

  Month = 2007 / 01

  Organization Name = Vision Switzerland
