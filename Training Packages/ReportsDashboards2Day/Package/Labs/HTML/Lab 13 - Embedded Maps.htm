         Lab Overview

         Open Analysis

         Add Map

Peak Indicators Limited

 

 

 

 

 

Lab Overview

 

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

         In this lab you will enhance an existing Analysis by adding a Map view

 

         There will be 3 Layers on the Map:

  Colour Fill

  Bar Graph

  Variable Marker

 

 

 

Information

 

      The completed Analysis will look like this:

 

 

 

 

 

 

Open Analysis

 

 

Peak Indicators Limited

 

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

 

 

 

Edit Analysis

 

         Open up the following Analysis for editing:

 

  Folder: Shared Folders > Lab Sessions

 

  Name: Sales Order Revenue by Business Group

 

 

 

 

 

 

 

 

 

 

 

 

 

Add Map View

 

         On the Compound Layout, add a Map view to the Analysis

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Default Map

 

         You should now see a Map view appear on the Analysis

 

         Edit the new Map view

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Map Properties

 

         Open up the Map properties and configure the Canvas as follows:

 

  Canvas Size: Custom

  Width: 700

  Height: 400

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Colour Fill

 

 

Peak Indicators Limited

 

 

 

 

 

Information

         The default Colour Fill layer on the Map needs to be configured:

  Change measure to Total Revenue instead of # Orders

  User Green / Amber / Red colouring

  Allow End Users to edit the Thresholds

 

 

 

 

 

 

 

 

Edit Colour Fill

 

         Choose the Edit option for the existing # Orders (Color Fill) layer on the Map view

 

 

 

 

 

 

 

Configure Colour Fill

 

         Configure the Colour Fill as follows:

  Vary Colour By: Total Revenue (Millions)

  Colours:

  Enable Allow Dashboard Users to Edit Thresholds

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Bar Graph

 

 

Peak Indicators Limited

 

Information

 

         You will now add an additional Bar Graph layer on top of the Map

 

         The Bar Graph will show Total Revenue figures with a bar for each Deal Size

 

 

 

Add Bar Graph

 

         Add a Bar Graph to the Map using the layer OBIEE_BUSINESS_GROUPS

 

 

 

 

 

 

 

 

Configure Bar Graph

 

         Configure the Bar Graph as follows:

  Bar Size: Total Revenue (Millions)

  Bars: Deal Size

 

 

 

 

 

 

 

 

 

 

Bubble

 

 

Peak Indicators Limited

 

 

 

 

 

Information

 

         Well now add a 3rd layer to the Map, this time it will be a Bubble where the size represents the # Orders for each Business Group

 

 

 

 

Add Bubble

 

         Add a Bubble to the Map using the layer OBIEE_BUSINESS_GROUPS

 

 

 

 

 

 

Edit Bubble

 

         Configure the Bubble as follows:

  Vary Size By: # Orders

 

(If it is not already selected)

 

 

 

 

 

 

 

 

 

Disable Bar Graph

 

         As the Bubble and Bar Graph overlap each other, disable the Bar Graph so that only the Colour Fill and Bubble are on display

 

 

 

 

 

 

 

 

 

Arrange Compound Layout

 

         Return back to the Compound Layout

         Arrange the layout so that the Map view is above the Pivot Table but underneath the Title view

 

 

 

 

 

 

 

 

 

Save Analysis

 

         Save the Analysis in your My Folders area, keeping the existing name:

  Sales Order Revenue by Business Group

 

 

 

 

 

 

 

 

 

Preview

 

         Preview your new Map Analysis!

 

 

 

 

 

 

 

 

 

Print to PDF

 

 

Peak Indicators Limited

 

 

 

 

 

 

 

 

Print to PDF

 

         At the bottom of the Map, click on the following option link:

  Print > Printable PDF

 

         A PDF document should now open containing the entire Analysis
