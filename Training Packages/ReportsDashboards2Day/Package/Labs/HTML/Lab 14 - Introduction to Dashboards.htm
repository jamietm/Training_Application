Lab Overview

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

      In this lab we will place a number of your Analyses onto 3 new pages within your personal My Dashboard

 

 

 

Information

 

      The first Dashboard Page will contain your Order Revenue by Quarter Analysis, it will be configured as follows:

  Page name will be Revenue by Quarter

  The page will consist of one Section, with a default border

  Collapsible button disabled

  Drill in Place enabled

 

 

 

 

Revenue by Quarter

 

Peak Indicators Limited

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Edit My Dashboard

 

      Go to your My Dashboard

 

      Open up the Dashboard Editor

 

 

 

 

 

 

 

 

Add Analysis

 

      Add the following Analysis to the empty Dashboard Page:

 

  Folder: My Folders

  Name: Order Revenue by Quarter

 

You should see a new Column and Section appear containing the Analysis

 

 

 

 


Configure Section Properties

 

      From the Section Properties:

 

  Enable the Drill in Place option

  Disable the Collapsible option

 

 

 

 

 

 

 

 

 

Rename Dashboard Page

 

      Open up the Dashboard Properties

 

      Rename the Dashboard Page to be: Revenue by Quarter

 

 

 

 

 

 

 

 

 

 

Save and Run

 

      Save and then run your first Dashboard Page!

 

      Observe that Navigation, Column Selectors and View Selectors work exactly as expected in the same way as they did within Answers

Service Status

 

Peak Indicators Limited

 

 

Information

 

      The layout of this final Dashboard Page will be slightly different, it will contain 3 Analyses giving an overview of the Service Incidents which are currently open:

  1 Column containing 3 Sections

  Each Section will have a Section Header and Title

  Section Borders disabled

  Collapsible option disabled

  Drill in Place option disabled

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Add Dashboard Page

 

      Edit to your My Dashboard

 

      Add a new Dashboard Page called: Service Status

 

 

 

 

 

 

 

Add Analysis

 

      Add the following Analyses to your new empty Dashboard Page, placing them in different Sections within the same Column:

 

  Folder: My Folders

  Name: Open Service Incidents by Type

 

  Folder: Shared > Service Incidents

  Name: High Severity Service Incidents Ticker

  Name: Open Service Incidents by Severity

 

 

 

 

 

 

 

 

Configure Section Properties

 

      For each of the 3 Sections, configure the Section Properties as follows:

 

  Enable the Drill in Place option

  Disable the Collapsible option

 

 

 

 

 

 

 

 

 

 

Save and Run

 

      Just to see the progress so far, save and then run your third Dashboard Page

 

 

 

 

 

 

 

 

 

 

 

Section 1 Header and Title

 

      Configure Section 1 to display a Header and Title:

  Enable Show Section Header option

  Enable Show Section Title option

 

      Rename Section 1 to be: Open Service Incidents by Type

 

 

 

 

 

 

 

 

 

 

Section 2 Header and Title

 

      Configure Section 2 to display a Header and Title:

  Enable Show Section Header option

  Enable Show Section Title option

 

      Rename Section 2 to be: High Severity Service Incidents Ticker

 

 

 

 

 

 

 

 

 

 

Section 3 Header and Title

 

      Configure Section 3 to display a Header and Title:

  Enable Show Section Header option

  Enable Show Section Title option

 

      Rename Section 3 to be: Open Service Incidents by Severity

 

 

 

 

 

 

 

 

 

Save and Run

 

      Save and run your third Dashboard Page!

 

 

 

 

 

 

 

 

 

 

Configure Dashboard Report Links

 

Peak Indicators Limited

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Configure Dashboard Report Links

 

      Edit the Dashboard Page once again and open up the Dashboard Properties

      Configure the Dashboard Properties so that the following links appear underneath all reports:

  Analyze Edit

  Refresh Print

  Export Add to Briefing Book

  Copy

 

 

 

 

 

 

 

 

Disable Dashboard Report Links on Service Status Page

 

      We dont actually want the Dashboard Report Links to be visible on the Service Status Dashboard Page, so configure each of the 3 Analyses on this page so that they do not show any Dashboard Report Links

 

      By doing this, you are overriding the default settings you previously made within Dashboard Properties

 

 

 

 

 

 

 

 

 

Configure Dashboard Report Links

 

 

Peak Indicators Limited

 

Information

 

      On the Service Status Dashboard Page, the third Analysis has two views displayed in its Compound Layout

 

      However showing two views is too much, so we will configure its Dashboard Section so that only the Graph view is displayed

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Show Graph View Only

 

      On the Service Status Dashboard Page, configure the bottom Analysis so that it only shows the Graph 1 view from the Analysis

 

 

 

 

 

 

 

 

 

 

Save and Run

 

      Save and run your Dashboard for the last time, you should notice all the Dashboard Report Links appearing at the bottom of the first two Dashboard Pages only:

  Revenue by Quarter

  Revenue by Business Group

      On the Service Status Dashboard Page, you should now only see the Graph view at the bottom, not the Map
