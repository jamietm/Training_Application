Lab Overview

 

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

         In this lab we are going to play the role of an End User who has access to a Dashboard Page but wants to analyse the content in a slightly different way

         The user will:

  Exclude certain members (Selections)

  Create custom groups (Groups)

  Create custom calculations (Calculated Items)

 

 

Information

 

         The completed Analysis will look like this:

 

 

 

 

 

 

Selections

 

 

Peak Indicators Limited

 

Information

 

         In this example, the End User will open up a Dashboard Page and then decide to modify the Analysis so that several Members are excluded from the results

         The Selections feature will be used to achieve this

 

 

 

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Product Categories

 

         Navigate to the following Dashboard:

  Sales Orders > Sales Orders > Product Categories

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Analyze

 

         At the bottom of the Dashboard Page, click on the Analyze link

  This will take you in to Answers so that you can further customise the Analysis on display

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Select Members

 

         On the Results tab, expand the Selection Steps pane

 

         Within the Product Category section, select the following option:

  Then, New Step > Select Members

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Remove Members

 

         Choose the Action Remove

 

         Search for and then select the following Members:

  Documentation

  Product Requirements

  Test Points

  Test, Tools & Supplies

  Test and Measurement

  Service Plans

 

 

 

 

 

 

 

 

 

 

 

 

 

View Selections

 

         View the Analysis, note that the list of Members has reduced as a result of the Selections you have just made

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Groups

 

 

Peak Indicators Limited

 

Information

 

      The End User has now decided to group the Members slightly differently

 

      An additional custom Group called Computers will be created that summarises the following Members together:

  Computer Parts and Components

  Computer Systems

  Software

 

 

 

 

 

Selection Steps

 

         Add another Selection Step to your Analysis, this time choose the following option:

  Add Groups or Calculated Items > New Group

 

 

 

 

 

 

New Group

 

         Configure your new Group as with the following Display Label:

  Computers

 

         Then select the following Members:

  Computer Parts and Components

  Computer Systems

  Software

 

 

 

 

 

 

 

Preview Group

 

         Preview your new custom Group!

 

 

 

 

 

 

 

 

 

 

Calculated Items

 

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

         The Analysis currently shows data by Quarter, however the user would like to see data summarised by Half Year

 

         We shall therefore create two Calculated Items to summarise the Quarters accordingly:

  2007 H1

  2007 H2

 

 

 

 

Selections Steps

 

         Add another Selection Step, but this time add it to the

Time Quarter section:

  Add Groups or Calculated Items > New Calculated Item

 

 

 

 

 

 

 

 

 

 

 

 

 

Configure Calculated Item

 

         Configure the Calculated Item as follows:

  Display Label: 2007 H1

  Function: Sum

 

         Then selected the following Members:

  2007 Q 1

  2007 Q 2

 

 

 

 

 

 

 

 

Selection Steps

 

         Add a second Selection Step to the Time Quarter section:

  Add Groups or Calculated Items > New Calculated Item

 

 

 

 

 

Configure Calculated Item

 

         Configure the Calculated Item as follows:

  Display Label: 2007 H2

  Function: Sum

 

         Then select the following Members:

  2007 Q 3

  2007 Q 4

 

 

 

 

 

 

Preview Results

 

         Preview the results of your two Calculated Items!

  Note that drill-down is not possible on Calculated Items
