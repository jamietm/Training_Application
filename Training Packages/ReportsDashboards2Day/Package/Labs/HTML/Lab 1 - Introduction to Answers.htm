Information

 

         The aim of this lab is to become familiar with the Oracle BI Answers by creating a new Analysis called Sales Ops Performance by Quarter

 

         The Analysis will consist of the following views:

  Title

  Table

  Graph

 

 

 

 

 

Information

 

      Once completed, your Analysis will look like this:

 

 

 

 

 

 

Create Analysis

 

 

Peak Indicators Limited

 

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

 

 

Create New Analysis

 

         Create a new Analysis using the Sales Orders Subject Area

 

 

 

 

 

 

 

 

 

 

Select Columns

 

         On the Criteria tab, select the following columns for your Analysis:

  Time.Quarter

  Facts.Order Amount

  Facts.Return Amount

  Facts.Cost of Sales

 

 

 

 

 

 

 

 

Rearrange Columns

 

         Rearrange the columns into the following order:

  Quarter

  Order Amount

  Cost of Sales

  Return Amount

 

         HINT: You can move a column to a new location simply by dragging it with the mouse to a new location

 

 

 

 

 

 

 

 

 

 

Add Filter

 

      Add the following filter to the Analysis

 

  Time.Year = 2007

 

 

 

 

 

 

 

 

 

Results Tab

 

         The criteria for your Analysis have now been defined!

 

         Go to the Results tab to display the default Compound Layout

 

 

 

 

 

 

 

Add Grand Total

 

         Edit the Table view

 

         Add a Grand Total to the Table

 

         Return back to the Compound Layout

 

 

 

 

 

 

 

 

 

Edit Title View

 

         Edit the Title view and specify the following settings:

  Title: Sales Ops Performance by Quarter

  Display Saved Name: [uncheck]

  Logo: fmap:images/logo.bmp

  Subtitle: Year 2007

 

         Return back to the Compound Layout

 

 

 

 

 

 

 

Add Vertical Bar Graph

 

         Add a new Vertical Bar Graph to the Analysis

 

 

 

 

 

 

 

 

 

 

Reduce Graph Size

 

         Edit the Graph view

 

         Reduce the size of the Graph by clicking on the Edit Graph Properties button and setting the following:

 

  Canvas Width : 600

  Canvas Height : 300

 

 

 

 

 

 

 

 

Configure Titles

 

         Now go to the Titles and Labels tab within the Graph Properties window

 

         Remove the Graph Title and Vertical Axis Title

  You can do this by clicking on the check-boxes underneath each corresponding text field

 

 

 

 

 

 

 

 

Abbreviate Vertical Axis

 

         Staying on the Titles and Labels tab, abbreviate the values displayed on the Graphs vertical axis into Millions:

 

  Click on the Vertical Axis Labels button

  Set Abbreviate to Millions(m)

 

 

 

 

 

 

 

 

Preview Analysis

 

         Return back to the Compound Layout

 

         Preview your Analysis

 

 

 

 

 

 

 

 

 

 

 

Save Analysis

 

         Save your Analysis in your My Folders area with the following name:

 

  Sales Ops Performance by Quarter

 

 

 

 

 

 

 

 

Open Analysis from Catalog

 

         Your Analysis is now complete!

 

         Go to the Catalog screen and open your new Analysis from within your My Folders area

 

 

 

 

 

 

 

 

 

 

Information

 

         In this lab you have learnt how to:

  Create a new Analysis

  Add a Grand Total to a Table view

  Configure a Title view

  Add a Bar Graph to an Analysis

  Save and Preview an Analysis
