Sales Order by Month

 

 

Peak Indicators Limited

 

 

Information

 

      We will now edit the Sales Orders by Month Analysis and replace its Table view with a Pivot Table

      We will also use the Pivot Table to replace the existing % Yearly Total calculation that is defined on the Criteria tab

      In addition, well use the Pivot Table to introduce a new calculation YTD # Orders which is calculated by doing a simple Running Sum of the # Orders measure

 

 

 

Information

 

      You final Pivot Table will look like this:

 

 

 

 

 

 

 

 

 

 

Open Analysis

 

      Open the following Analysis for editing:

 

  Folder: My Folders

 

  Name: Sales Orders by Month

 

 

 

 

 

 

 

 

Delete % Yearly Total

 

      On the Criteria tab, delete the % Yearly Total column from the Analysis

 

  NOTE: We will re-implement this calculation on the Pivot Table

 

 

 

 

 

 

 

 

Remove Table View

 

      Go back to the Results tab and remove the Table view from the Compound Layout

 

 

 

 

 

 

 

 

New Pivot Table

 

      Add a new Pivot Table to the Compound Layout

 

      Rearrange the Compound Layout so that the new Pivot Table is located directly underneath the Title view (and above the Graph)

 

 

 

 

 

 

 

 

Exclude Year Column

 

      Staying on the Compound Layout, exclude the Year column from the Pivot Table view

 

 

 

 

 

 

 

 

 

 

 

Swap Rows and Columns

 

      Using a single move of the mouse, swap the Measure Labels (across the top of the Pivot Table)) with the Month down the left

 

  HINT: You can achieve this by dragging the Measure Labels directly on top of the Month column

 

 

 

 

 

 

 

 

 

 

Add % Yearly Total Layer

 

      Edit the Pivot Table view

      Use the Duplicate Layer option to duplicate the existing # Orders measure

  You will see a second # Orders measure appear at the end

 

      Rename this new layer to: % Yearly Total

 

 

 

 

 

 

 

 

 

 

 

Apply % Yearly Total Calculation

 

      Now apply the % Yearly Total calculation by modifying the layers properties as follows:

 

  Show Data As > Percent of > Row

 

 

 

 

 

 

 

 

Create YTD # Order Layer

 

      Duplicate the # Orders measure again

 

      This time, rename the new layer to be: YTD # Orders

 

 

 

 

 

 

 

 

 

 

Apply YTD # Orders Calculation

 

      Apply the YTD # Orders calculation by simply enabling the following property for the new layer:

 

  Display as Running Sum

 

 

 

 

 

 

 

 

 

Preview and Save

 

      Return back to the Compound Layout

      Preview the Analysis to make sure all changes have been applied correctly

      Save the Analysis

 

 

 

 

 

 

 

 

 

 

 

 

 

Order Revenue by Quarter

 

 

Peak Indicators Limited

 

 

Information

 

      We will now create a new Analysis called Order Revenue by Quarter

 

      The Analysis will be configured to have a Pivot Table and a nice set of charts

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Create New Analysis

 

      Create a new Analysis using the Sales Orders Subject Area

 

 

 

 

 

 

 

 

 

Select Columns

 

      Add the following 3 columns to your Analysis:

  Time.Quarter

  Organization.Business Group

  Facts.Total Revenue (Millions)

 

 

 

 

 

 

 

 

 

Add Filter

 

      Add the following filter to the Analysis:

 

  Time.Year = CURR_YEAR (Repository Variable)

 

 

 

 

 

 

 

 

 

Remove Table View

 

      Go to the Results tab and remove the Table view from the Compound Layout

 

 

 

 

 

 

 

 

 

Create Pivot Table

 

      Add a new Pivot Table to the Compound Layout

 

 

 

 

 

 

 

 

 

Re-arrange Dimensions

 

      Using the mouse, move the Quarter dimension so that it is located underneath Total Revenue (Millions) across the top of the Pivot Table

 

 

 

 

 

 

 

 

 

Add Grand Total

 

      Edit the Pivot Table view

 

      Add a Grand Total to the end of the Rows section of the Pivot Table

 

 

 

 

 

 

 

 

 

Add Bar Chart

 

      Return back to the Compound Layout

 

      Add a Vertical Bar Chart to your Analysis

 

 

 

 

 

 

 

 

 

Move Quarter to Sections

 

      Edit your Bar Graph

  You should now see a separate Bar Graph for each of the four Quarters that appear in the Analysis

 

 

 

 

 

 

 

 

 

Display as Slider

 

      Now enable the Display as Slider option in the Sections area

 

      You should observe that your Bar Graph now has a Slider above it that allows you to scroll through the various Quarters that are available

 

      Click the button to see each of the Quarters being displayed in an automated sequence

 

 

 

 

 

 

 

 

 

Group By and Vary Colour

 

      Now move Quarter in to the Group By section

 

      Then move Business Group in to the Vary Color By Section

 

 

 

 

 

 

 

 

Stacked Bar Chart

Modify Properties

      Return back to the Compound Layout
