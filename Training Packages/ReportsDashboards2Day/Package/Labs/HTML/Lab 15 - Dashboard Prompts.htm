Information

 

      In this lab we are going to build a Dashboard Prompt for your Revenue by Quarter Dashboard Page. It will contain the following columns:

  Time (Defaults to CURR_YEAR)

  Quarter (Limited by Year)

  Business Group

  Total Revenue (Millions) (Slider)

 

 

 

Information

 

      In order for the Dashboard Prompt filters to take effect, a set of Is Prompted filters must be applied to the Analyses on the Dashboard Page. In our case we already have a Saved Filter called Is Prompted Filters which we can use.

      However, this Saved Filter does not include an Is Prompted filter on the column Total Revenue (Millions), so we will need to add one before applying the Saved Filter to our Revenue by Quarter Analysis

Create Dashboard Prompts

 

 

Peak Indicators Limited

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Create Dashboard Prompt

 

      Create a new Dashboard Prompt using the Sales Orders Subject Area

 

 

 

 

 

 

 

Year Prompt

 

      Create a new Column Prompt on the Time.Year column and configure it as follows:

  User Input: Choice List

 

  Choice List Values: Specific Column Values, 2000-2010

 

  Disable option Enable user to select multiple values

 

  Default Selection: Server Variable, CURR_YEAR

 

 

 

 

 

 

 

 

 

Quarter Prompt

 

      Create a new Column Prompt on the Time.Quarter column and configure it as follows:

  User Input: Check Boxes

 

  Limit values by: Year

 

 

 

 

 

 

 

 

 

 

 

Business Group Prompt

 

      Create a new Column Prompt on the Organization.Business Group column and configure it as follows:

  User Input: Check Boxes

 

  Check Boxes Width: Dynamic

 

 

 

 

 

 

 

 

 

 

Total Revenue (Millions) Prompt

 

      Create a new Column Prompt on the Facts.Total Revenue (Millions) column and configure it as follows:

  Operator: is greater than

  User Input: Slider

  Lower Limit: 0

  Upper Limit: 500

  Show Spin Box: [Enable]

  Compress Values: [Enable]

  Slider Orientation: Vertical

  Slider Size: Medium

 

 

 

 

 

 

 

 

 

Preview Dashboard Prompt

 

      Preview the Dashboard Prompt

 

      Observe that when you select a new value in the Year prompt the Quarter prompt values automatically change!
