Lab Overview

 

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

      In the previous lab you created a Sales vs Forecast Analysis

      We are now going to enhance this Analysis by enabling Master-Detail Linking, so that when you click on one of the Gauges the Bar Graph underneath will change to display data for the same Quarter as the Gauge

      Note that Master-Detail Linking only works on Views which have a Slider or Pages sections configured

 

 

Information

 

      Your completed Analysis will look like this:

 

 

 

 

 

Open Analysis

 

 

Peak Indicators Limited

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

Open Analysis

 

      Open up the following Analysis for editing:

 

  Folder: My Folders

 

  Name: Sales vs Forecast

 

 

 

 

 

 

 

Enable Master-Detail Channel

 

      On the Criteria tab, open up the Column Properties for the Forecast % column

  NOTE: Forecast % is the measure used on the Gauges

 

      Configure the column value to Send Master-Detail Events over the following channel:

  Forecast

 

 

 

 

 

 

 

 

 

Add Slider to Bar Graph

 

      Now go to the Results tab

 

      Edit the Bar Graph view

 

      Configure the Bar Graph to have a Slider on the Quarter column

 

 

 

 

 

 

 

 

 

Listen to Master-Detail Channel

 

      Open up Graph Properties

 

      Configure the Bar Graph to listen for Master-Detail Events over the following channel:

  Forecast

 

 

 

 

 

 

 

 

Preview and Save

 

      Return to the Compound Layout

 

      Preview the Analysis and test out the Master-Detail Linking!

  You should be able to click on one of the Gauges and watch the Bar Graph change to show data for the same Quarter

 

      Save your Analysis

 

 

 

 

 

 

 

 

Information

 

      In this lab you have learnt how to enable Master-Detail Linking
