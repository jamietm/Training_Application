Information

 

      In this lab we will create a new Analysis called Order Revenue by Product Category

      Product Category happens to be a parent-child hierarchy, so we will use a Hierarchical Column within the Analysis to allow us to drill-through the Product Category hierarchy

      Well also add a Pie Graph to the Analysis which is also based on the Hierarchical Column

 

 

 

 

Information

 

      Your completed Analysis will look like this:

 

 

 

 

 

 

 

 

Create New Analysis

 

 

Peak Indicators Limited

 

 

 

 

 

 

 

 

 

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Create Analysis

 

      Create a new Analysis using the Sales Orders Subject Area

 

 

 

 

 

 

 

 

Select Columns

 

      Add the following 3 columns to your Analysis, by double clicking or dragging them from the Subject Area:

  Product.Product Category.Product Category

  Time.Quarter

  Facts.Total Revenue (Millions)

 

      NOTE: The 1st column listed is a Hierarchical Column

 

 

 

 

 

 

 

 

Add Filter

 

      Add the following filter to the Analysis:

 

  Time.Year = CURR_YEAR (Repository Variable)

 

 

 

 

 

 

 

 

 

 

 

Configure Pivot Table Columns

 

      Go to the Compound Layout

  NOTE: By default, you now have a Pivot Table instead of a standard Table view

 

      Restructure the Pivot Table so that the Quarter column is now located across the top of the Pivot Table (but underneath the Total Revenue (Millions) measure column)

 

 

 

 

 

 

 

 

 

Expand Hierarchy

 

      At the moment the entire Product Category hierarchy is collapsed, so expand the node Vision High-Tech to see all its child nodes

 

 

 

 

 

 

 

 

Add Pie Graph

 

      Add a new Pie Graph to the Analysis

 

 

 

 

 

 

 

 

Expand Pie Graph

 

      NOTE: Just like the Pivot Table, the Pie Graph starts collapsed at the very top-level node, so the Pie will just be one single colour representing the whole of Vision High-Tech

 

      Click once on the Pie Graph to expand it out to its child nodes

 

 

 

 

 

 

 

 

Edit Title View

 

      Return to the Compound Layout

      Configure the Title view as follows:

  Title: Order Revenue by Product Category

  Display Saved Name: [uncheck this option]

  Logo: fmap:images/logo.bmp

  Subtitle: Figures displayed in millions

 

 

 

 

 

 

 

 

 

Preview Analysis

 

         Preview the Analysis

 

         Test the Analysis out by expanding and collapsing on a couple of columns

 

         You could also try switching the Product Category Hierarchical Column with the Quarter column

 

 

 

 

 

 

 

 

 

Save Analysis

 

         Save the Analysis as the following:

 

  Folder: My Folders

 

  Name: Order Revenue by Product Category
