Lab Overview

 

 

Peak Indicators Limited

 

 

 

 

 

 

Information

 

      Were now going to look at a number of additional views which can be used to enhance your Analyses even further. All the features in this lab are focussed on:

  Improving usability and intuitiveness for the end users

  Increasing interactivity

  Reducing the number of objects on display without losing functionality

 

 

 

Information

 

      First of all, we are going to edit the Order Revenue by Quarter Analysis and implement a Column Selector view

      The Column Selector will enable the user to choose different Dimension and Measure combinations across all the views on the Analysis

 

 

 

 

 

Column Selector

 

 

Peak Indicators Limited

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Log on as your Power User account

 

         Log on to Oracle BI EE 11g:

  http://obiee11g:7001/analytics

 

         Use your Power User account credentials. For example:

  User Id: poweruserxx

  Password: welcome1

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Open Analysis

 

      Open the following Analysis for editing:

 

  Folder: My Folders

 

  Name: Order Revenue by Quarter

 

 

 

 

 

 

Add Column Selector

 

      On the Compound Layout, add a Column Selector view to the Analysis

 

      Rearrange the Compound Layout so that the Column Selector is positioned directly underneath the Title view (and above the Pivot Table)

 

 

 

 

 

 

 

 

Configure Column Selector

 

      Edit the Column Selector view

      Choose to include a Selector for the Business Group column

      Specify the caption: Choose Dimension

      Select the following columns to list in the Column Selector:

  Customer.Sales Channel Code

  Customer.Customer Class Code

  Order.Deal Size

 

 

 

 

 

 

 

 

 

Configure Next Selector

 

      Now choose to include a Selector for the Total Revenue (Millions) column

      Specify the caption: Choose Measure

      Select the following columns to list in the Column Selector:

  Fact.# Orders

  Fact.Sales Margin %

 

 

 

 

 

 

 

 

Configure Sales Margin %

 

      Configure the data format of the Sales Margin % column so that values are displayed as follows:

 

  Treat Numbers As: Percentage

 

  Decimal Places: 1

 

 

 

 

 

 

 

 

 

Test

 

      Return back to the Compound Layout and try out your new Column Selector! For example:

  Dimension: Deal Size

  Measure: Sales Margin %

See that the Graphs change as well as the Pivot Table! Also note that the Sales Margin % measure should be displayed as a percentage with 1 decimal place

      Save your Analysis

 

 

 

 

 

 

 

 

 

 

View Selectors

 

 

Peak Indicators Limited

 

 

 

Information

 

      The Order Revenue by Quarter Analysis which you have just been editing has three Graph views displayed on the Compound Layout

 

      There are too many views now to fit on a single screen, so we will now look at implementing a View Selector so that there is only one Graph visible at any time and the user can choose which one to display

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Remove Existing Graph

 

      Remove the existing Bar Graph, Pie Graph and Waterfall Graph from the Compound Layout

 

 

 

 

 

 

 

 

 

Add View Selector

 

      Add a new View Selector to the Compound Layout

 

      Edit the new View Selector

 

 

 

 

 

 

 

 

Configure View Selector

 

      Specify the caption: Choose View

 

      Include the following three views for the View Selector:

  Graph

  Graph:2

  Graph:3

 

      Click Done to return to the Compound Layout

 

 

 

 

 

 

 

 

 

 

Rename View

 

      To make it more user-friendly for the end user, rename the selected views as follows:

 

  Graph: Bar Graph

  Graph:2: Pie Graph

  Graph:3: Waterfall Graph

 

 

 

 

 

 

 

 

 

Test and Save

 

      Test your new View Selector!

 

      Save the Analysis

 

 

 

 

 

 

 

 

 

 

 

 

Gauges

 

 

Peak Indicators Limited

 

 

Information

 

      We will now look at enhancing the Sales vs Forecast Analysis, replacing the existing Pivot Table with a set of Gauges to provide better visualisations

 

 

 

 

Information

 

      Your final Analysis will look like this:

 

 

 

 

 

 

 

 

 

 

Open Analysis

 

      Open the following Analysis for editing:

 

  Folder: My Folders

 

  Name: Sales vs Forecast

 

 

 

 

 

 

 

 

 

 

Remove Pivot Table, Add Gauge

 

      Remove the Pivot Table view from the Compound Layout

 

      Add a Gauge view to the Compound Layout

 

      Rearrange the Compound Layout so that the Gauge view is located just underneath the Title view

 

 

 

 

 

 

 

 

Configure Gauge

 

      Edit the Gauge view

 

      Exclude the Business Group column so that only Quarter is left in the Rows section

 

 

 

 

 

 

 

 

 

Gauge Properties - General

 

      Open up the Gauge Properties and configure settings on the General tab as follows:

 

  Gauges per Row: 4

 

 

 

 

 

 

 

 

 

Gauge Properties - Scale

 

      Configure the Scale tab of the Gauge Properties so that the Gauges show a range of 0 - 120:

 

  Gauge Limits: Minimum 0

  Gauge Limits: Maximum 120

 

 

 

 

 

 

 

 

Gauge Properties - Titles and Labels

 

      On the Titles and Labels tab of Graph Properties, configure as follows:

 

  Footer: @1

(This denotes column 1 i.e. Quarter)

 

  Scale Labels: Actual Value

 

 

 

 

 

 

 

 

Adjust Size

 

      Change the size of the Gauges to Medium instead of Fit Canvas

 

 

 

 

 

 

 

 

Status Labels

 

      Expand the Settings window pane at the very bottom of the screen

      There are 3 different Status threshold values, change them to be the following:

  Best Forecast

  Approaching Forecast

  Warning

 

      HINT: Click on the button and choose the Specify Label option

 

 

 

 

 

 

 

 

 

Thresholds

 

      There are two different threshold values to configure, set them to the following:

  100

  90

 

      HINT: Click on the button and choose the Custom Value option

 

 

 

 

 

 

 

 

Preview and Save

 

      Return to the Compound Layout

 

      Preview the Analysis to confirm it is configured correctly

 

      Save the Analysis

 

 

 

 

 

 

 

 

 

 

 

Static Text and Narrative Views

 

 

Peak Indicators Limited

 

 

Information

 

      We are now going to edit another Analysis called Open Service Incidents by Type

      Instead of showing a very basic Title and Table view, we will instead use a Static Text and Narrative View to provide a more interesting result:

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Open Analysis

 

      Open the following Analysis for editing:

 

  Folder: /Shared/Lab Sessions

 

  Name: Open Service Incidents by Type

 

 

 

 

 

 

 

 

 

Remove Table View

 

      Remove the Table View from the Compound Layout

 

 

 

 

 

 

 

 

Add Static Text View

 

      Add a new Static Text view to the Compound Layout

 

      Edit the Static Text view

 

 

 

 

 

 

 

 

Configure Static Text View

 

      Add the following text into the Static Text view, to show a caption followed by the system time and date:

 

  Current Date and Time : @{system.currentTime}

 

 

 

 

 

 

 

 

 

Format Caption

 

      Format the caption Current Date and Time into bold

  You can do this by highlighting the text and then clicking on the button

  You will see a [b] and [/b] tags appear before and after the text

 

      Return to the Compound Layout

 

 

 

 

 

 

 

 

 

 

Add Narrative View

 

      Add a Narrative View to the Compound Layout

 

      Edit the Narrative View

 

 

 

 

 

 

 

 

 

Configure Narrative Text

 

      Paste the following text into the Narrative part of the view, this will display Column 1 followed by Column 2 from the Analysis:

  @1: @2

 

      Copy the following into the Row Separator to separate out the records:

  .........

 

 

 

 

 

 

 

 

 

Text Format

 

         Format the @1 text in the Narrative view to be bold

 

 

 

 

 

 

 

 

 

Preview and Save

 

      Return to the Compound Layout

 

      Preview the Analysis

 

      Save the Analysis as the following:

  Folder: My Folders

  Name: Open Service Incidents by Type

Information

 

      We will now enhance the Sales Performance Analysis by adding a Trellis View

 

 

 

Remove/Add Views

 

         Go to the Results tab and remove the graph view and table view

 

         Add an Advanced Trellis

 

 

 

 

 

 

 

 

 

 

Edit Trellis View

 

         Edit the Trellis View

 

         Arrange the layout of the Trellis View as follows:

  Columns: Business Group

  Rows: Customer Class Code

  Measures: # Orders

  Bars: Year

  Excluded: All others

 

 

 

 

 

 

 

 

 

Increase Graph Size

 

         Open up Graph Properties

 

         In the Visualization section, change Graph Size to Custom

 

         Change the Height to 60 pixels

 

         Return to the Compound layout

 

 

 

 

 

 

 

 

 

 

Rearrange Trellis View

 

         In the Compound Layout, drag the Business Group heading over the # Orders row on the Trellis View

 

 

 

 

 

 

 

 

 

 

 

 

 

Preview Results

 

         Click Preview to see your new Trellis View!

 

 

 

 

 

 

 

 

 

 

 

 

 

Save Analysis

 

         Save the Analysis as the following:

 

  Folder: My Folders

  Name: Sales Performance Analysis - Trellis
